Projects

ExposureVision

Dow Chemical Internship

ExposureVision is a web application that gives clients and stakeholders a modern user interface to monitor over 30,000 paint panels being tested worldwide. My biggest achievement was coding re-usable widgets for displaying customizable panel data for the main dashboard, which other developers were then able to use ‘out of the box’ in other parts of the application.

Dow Promo Video

MapsPlay

Personal Project

MapsPlay is a game in which the user finds themselves in a random Google Street View instance around the world and must guess their location. I am especially proud of the Javascript in this application, which uses a Singleton based design pattern to make clean modular code. The application took me roughly 4 months to complete in my spare time, and is completely responsive on mobile browsers.

Pinteresting

Personal Project

Pinteresting is a pinterest clone that I created in my spare time with Ruby on Rails. Some of the things I am most proud of with this application are the responsive design, MVC architecture, and management of user sessions. Recently I have used this site as an experiment with SEO and was able to get it to the front page of google for several of my key phrases.
